## 🗂️ Visual C++ / Visual Studio Folder Structure (Typical)

```bash
.
├── .vs/                 # Visual Studio workspace settings
├── ProjectName/        # Visual C++ project folder
│   ├── ProjectName.sln # Solution file
│   ├── ProjectName.vcxproj
│   ├── ProjectName.vcxproj.filters
│   ├── src/            # Source files (.cpp)
│   │   └── main.cpp
│   ├── include/        # Header files (.h / .hpp)
│   └── x64/            # Build outputs (Debug/Release)
│       ├── Debug/
│       └── Release/
